# ANI - Rule the chat

ANI is an open-source real-time chat application built with React and Node.js. It provides a seamless chatting experience with features like user authentication, real-time messaging, and a user-friendly interface with most important feature of privacy based communication for users.

## Features

- Real-time messaging
- User authentication 
- Private group and direct messaging
- User-friendly interface
- Modern Ui design
- Open-source and customizable

## Tech used

- React
- Node.js
- Socket.io
- Express
- MongoDB
